## 项目特点

优点：地理位置优越，宿舍价格在纽约属于便宜的一档，课程内容整体比较新颖，class size 小。

缺点：一年的项目，不可延期，没法实习。伊萨卡 cs meng 可以延期，康泰克 CM 2 年可以实习。

备注：Meng CS两个学期都有Studio，第一学期的Studio是选课题然后系统分配，第二学期的Studio是自己找人组队。实际上Studio对个人无论学术还是职业的帮助都相对比较有限。
